Stuff I've done...

Zipline DMP - i-Behavior

The work: Build and refactor the dmp.ibzipline.com web application.

The challenge: We inherited this code-base from a 3rd party and are working diligently to replace the current old-school post-back architecture with RESTful services, Laravel as our back end PHP framework, and AngularJS/JSON on the front-end.

The technology: HTML, CSS(3), Foundation, PHP, JavaScript, AngularJS, jQuery

snow.com Home Page Re-work - Vail Resorts

The work: Re-work the www.snow.com home page to replace flash with HTML while maintaining similar functionality.

The challenge: There was a lot of marketing content that had to be built for this project and managing that in Sitecore (CMS) proved problematic. There was also a lot of functionality that existed in our other sites that had to be extended to work on snow.com.

The technology: HTML, CSS(3), C#/.Net, JavaScript, jQuery, Sitecore, Photoshop

Breckenridge and Heavenly Site Re-skins - Vail Resorts

The work: Re-skin www.breckenridge.com and www.skiheavenly.com. I worked with/led two contract developers on a full re-skin of both sites over a period of three months.

The challenge: We weren't given the time to refactor/change any of the HTML or JavaScript and had to do nearly all of the work in CSS.

The technology: Almost all CSS(3) and Photoshop with a little bit of HTML, JavaScript, jQuery, Sitecore, and C#/.Net

Lift Ticket Search and Results Pages Rebuild - Vail Resorts

The work: Re-build the lift ticket search and lift ticket results pages for all resorts ( Vail, Beaver Creek, Breckenridge, Keystone, Heavenly, and Snow).

The challenge: We only had comps for Vail and Snow so I had to use my creative noggin to pick colors, fonts, and layouts that worked with the design of the rest of the sites. There was also a very big analytics effort and there had to be a lot of JavaScript created to handle the Omniture tracking.

The technology: HTML, CSS(3), JavaScript, jQuery, C#/.Net, Sitecore, Omniture, REST/AJAX, Photoshop

epicpass.com Season Pass Page - Vail Resorts

The work: Re-build the Snow.com Season Pass page so more than 4 passes would fit.

The challenge: This was my first project using jQuery and I had to learn quickly!

The technology: HTML, CSS(3), C#/.Net, JavaScript, jQuery, Sitecore, Omniture, Photoshop
